Exercice numéro 1

Ligne de temps faite avec Wikidata

Cette timeline montre les acteurs dont la langue maternelle est l'italien.

La requête est :

#defaultView:Timeline SELECT ?painter ?painterLabel ?date_of_birth ?place_of_birth ?place_of_birthLabel ?image ?coordinate_location WHERE { SERVICE wikibase:label { bd:serviceParam wikibase:language "[AUTO_LANGUAGE],en". } ?painter wdt:P106 wd:Q33999; wdt:P103 wd:Q652. OPTIONAL { ?painter wdt:P569 ?date_of_birth. } OPTIONAL { ?painter wdt:P19 ?place_of_birth. } ?painter wdt:P18 ?image. } LIMIT 100
